This guide details how I configured my laptop in order to work with [Puppet](https://www.puppet.com/)

In order to create Puppet scripts in VSCode it is recommended to use the following extension which provides recommendations, syntax highlighting, and info-on-hover:

- [puppet.puppet-vscode](https://marketplace.visualstudio.com/items?itemName=puppet.puppet-vscode)

The extension requires the following packages to be installed:

```sh
brew install --cask puppetlabs/puppet/pdk
```
