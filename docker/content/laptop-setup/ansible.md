This guide details how I configured my laptop in order to work with [Ansible](https://docs.ansible.com/)

In order to create Ansible scripts in VSCode it is recommended to use the following extension which provides recommendations, syntax highlighting, and info-on-hover:

- [redhat.ansible](https://marketplace.visualstudio.com/items?itemName=redhat.ansible)

The extension requires the following packages to be installed:

```sh
brew install ansible ansible-lint ansible-language-server
```
