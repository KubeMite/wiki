This guide details how I configured my laptop in order to work with [Docker](https://www.docker.com/)

Download Docker desktop: [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

In order to create Dockerfiles in VSCode it is recommended to use the following extensions which provides recommendations, syntax highlighting, and info-on-hover:

- [ms-azuretools.vscode-containers](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-containers)
- [exiasr.hadolint](https://marketplace.visualstudio.com/items?itemName=exiasr.hadolint)

The extension requires the following packages to be installed:

```sh
brew install hadolint
```
